# Demo Script

Target length: 3 minutes.

## Opening

"AI agents can discover and buy services, but handing an LLM an unrestricted wallet is not acceptable. CSPR Sentinel gives agents bounded autonomy through deterministic policy and verifiable reputation."

## Live flow

1. Show the active policy: 0.05 WCSPR automatic limit, 0.50 daily budget, and provider trust threshold.
2. Start the RWA diligence run for the sample commercial paper instrument.
3. Show Asset Verification settle automatically for 0.02 WCSPR.
4. Open its Casper Testnet transaction in the explorer.
5. Show Risk Intelligence waiting in the approval queue at 0.12 WCSPR.
6. Explain that no payment signature exists yet.
7. Approve the request and show the delivered result and transaction hash.
8. Submit the verified-buyer rating.
9. Show provider and agent reputation counters plus the contract anchor.
10. End on the MCP endpoint and list the five tools available to any compatible agent.

## Closing

"CSPR Sentinel lets an agent act economically without removing human control. Casper provides the settlement evidence, and bilateral reputation turns each verified outcome into safer future decisions."
